A modern MATLAB toolbox for dipolar EPR spectroscopy
Download 0.8.beta Documentation

Comprehensive

DeerLab includes most popular dipolar analysis methods as well as more modern techniques.

Easy to use

DeerLab scripting is accessible for programmers of any experience level.

Flexible

A single interface for a wide range of applications: ranging from simple data analysis to development and testing of new methodology.

Open source

All of DeerLab's source code is publicly available on Github and provided under the MIT license.
