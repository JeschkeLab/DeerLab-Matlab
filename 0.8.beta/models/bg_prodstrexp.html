bg_prodstrexp¶

Product of two stretched exponentials background parametric model

Syntax¶

info = bg_prodstrexp()
P = bg_prodstrexp(r,param)
Parameters
  • t - Time axis (N-array)
  • param - Model parameters
Returns
  • B - Model background (N-array)
  • info - Model information (struct)

Model¶

Variable Symbol Default Lower bound Upper bound Description
param(1) 3.5 0 200 1st strexp decay rate
param(2) 3 0 6 1st strexp fractal dimension
param(3) 3.5 0 200 2nd strexp decay rate
param(4) 3 0 6 2nd strexp fractal dimension

Description¶

info = bg_prodstrexp()

Returns an info structure containing the specifics of the model:

  • info.model - Full name of the parametric model.
  • info.nparam - Total number of adjustable parameters.
  • info.parameters - Structure array with information on individual parameters.
B = bg_prodstrexp(t,param)

Computes the background model B from the axis t according to the parameters array param. The required parameters can also be found in the info structure.
